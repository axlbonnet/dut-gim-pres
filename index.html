Un problème simple

Calcul d'un score

  • Obtenez le score de votre nom en additionant les longueurs de vos prénoms et de vos noms
  • Obtenez le score de votre anniversaire en additionant le jour, le mois et l'année de votre naissance
  • Multipliez ces deux scores pour obtenir votre score de base
  • Votre score final est le score de base modulo 37

Code C calcul de score


#include <stdio.h>

int main(void) {
        int scoreNom, scoreAnniversaire;
        int scoreFinal;

        scoreNom = 4 + 6;
        scoreAnniversaire = 15 + 4 + 1997;
        scoreFinal = scoreNom * scoreAnniversaire;
        scoreFinal = scoreFinal % 37;
        printf("Ton score : %d \n", scoreFinal);
        return 0;
}

Code C calcul de l'aire d'un triangle (valeurs fixes)


#include <stdio.h>

int main(void)
{
        float longueur, largeur;
        float surface;

        longueur = 3.1;
        largeur = 5.8;

        surface = (longueur * largeur) / 2;

        printf("surface %f \n", surface);
        return 0;
}

Code C calcul de l'aire d'un triangle (valeurs saisies)


#include <stdio.h>

int main(void)
{
        float longueur, largeur;
        float surface;

        printf("Entrez la longueur du triangle :");
        scanf("%f", &longueur);
        printf("Entrez la largeur du triangle :");
        scanf("%f", &largeur);

        surface = (longueur * largeur) / 2;

        printf("surface %f \n", surface);
        return 0;
}

Les conditions

Expression qui ne peut avoir que deux valeurs (aka Booléen) :

  • Vrai
  • Faux

Construite avec des opérateurs relationels : > < ≤ ≥ = ≠

Ou en combinant des conditions : non / ou / et
